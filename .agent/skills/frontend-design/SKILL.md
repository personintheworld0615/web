# Frontend Design

## Core Philosophy

Before writing a single line of code, establish a **BOLD aesthetic direction**. Consider:
- What is the purpose and context of this interface?
- Who is the audience and what tone should it convey?
- What technical constraints exist?
- **What makes this UNFORGETTABLE?**

Choose a clear conceptual direction and execute it with precision.

## Design Pillars

### Typography
- Choose fonts that are **beautiful, unique, and interesting**
- Avoid generic fonts like Arial and Inter
- Opt for distinctive choices that elevate the frontend's aesthetics
- Use type as a design element, not just a content carrier
- Establish a clear typographic hierarchy

### Color
- Build cohesive color themes using CSS variables
- Use a dominant color with sharp, intentional accents
- Avoid predictable schemes (e.g. purple gradients on white backgrounds)
- Design for contrast, mood, and brand resonance

### Motion
- Use CSS animations and scroll-triggered effects
- Orchestrate animations strategically — delight at specific moments, not scattered micro-interactions
- Match animation complexity to the aesthetic vision
- Motion should feel purposeful, not decorative noise

### Spatial Composition
- Embrace asymmetry, overlap, and unexpected layouts
- Break the grid intentionally and with confidence
- Use whitespace as a design tool
- Create visual tension and resolution

### Atmosphere
- Use backgrounds, textures, and visual details to reinforce the aesthetic
- Layer depth with shadows, gradients, and blur
- Every visual detail should serve the conceptual direction

## What to Avoid

- Generic AI-generated aesthetics
- Overused or "safe" typefaces
- Clichéd color schemes
- Predictable, cookie-cutter layouts lacking contextual character
- Scattered micro-interactions without strategic purpose
- Complexity for its own sake

## Implementation Principle

**Match implementation complexity to vision.**

- Maximalist designs require elaborate code with extensive effects
- Minimalist designs demand restraint and precision in spacing and typography
- Elegance comes from executing the vision well — not from adding more

Always ask: *Is this choice serving the design direction, or is it a default?*
